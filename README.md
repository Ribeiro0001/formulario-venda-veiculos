# Formulário de Venda de Veículos

Projeto desenvolvido para praticar formulários utilizando HTML5 e CSS3.

## 🚀 Tecnologias

- HTML5
- CSS3

## 📖 Sobre

Aplicação com formulário de cadastro de veículos contendo diversos tipos de inputs, upload de imagens, radio buttons e checkboxes.

## 🎯 Objetivo

Praticar a criação e estilização de formulários utilizando HTML e CSS.
